# Spooky Agent 👻

A local AI agent powered by Gemini, designed to perform tasks, manage knowledge, and interact with a voice and visual presence.

## Getting Started
- Planning is located in [PLAN.md](./PLAN.md).
- The workspace is set up for Python development.

## Setup
1. Create a virtual environment: `python3 -m venv venv`
2. Activate it: `source venv/bin/activate`
3. Install dependencies: `pip install google-generativeai mcp fastapi uvicorn` (Coming soon)
